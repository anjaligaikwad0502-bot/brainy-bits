Expense Tracker
A simple Python-based console application to track daily expenses. Users can add new expenses with category and notes, view all recorded expenses, and calculate total spending. Data is stored in a local text file (expenses.txt) for easy retrieval.

Features:
1.Add new expenses with amount, category, and notes
2.View all recorded expenses in a readable format
3.Calculate total expenses
4.Persistent storage using a text file

Tech Stack:
Python 3
File handling (text file for storage)
